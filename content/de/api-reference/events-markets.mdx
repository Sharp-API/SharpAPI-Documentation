# Event-Märkte

Rufen Sie die verfügbaren Markttypen für ein bestimmtes Event ab, einschließlich der Anzahl der Sportsbooks, die jeden Markt anbieten.

```
GET /api/v1/events/{eventId}/markets
```

## Authentifizierung

Erfordert einen API key über den `X-API-Key` Header, den `Authorization: Bearer` Header oder den `api_key` Query-Parameter. Verfügbar für alle Tarife.

## Pfadparameter

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|------|----------|-------------|
| `eventId` | string | Ja | Eindeutiger Event-Bezeichner (z. B. `evt_nba_bos_lal_20260208`) |

## Antwort-Header

Alle Antworten enthalten standardmäßige Rate-Limit- und Metadaten-Header:

| Header | Beschreibung |
|--------|-------------|
| `X-RateLimit-Limit` | Maximale Anfragen pro Minute für Ihren Tarif |
| `X-RateLimit-Remaining` | Verbleibende Anfragen im aktuellen Zeitfenster |
| `X-RateLimit-Reset` | Unix-Zeitstempel, wann das Rate-Limit-Fenster zurückgesetzt wird |
| `X-Data-Delay` | Datenverzögerung für Ihren Tarif (z. B. `0s`, `60s`) |
| `X-Request-Id` | Eindeutige Anfrage-ID zur Fehlerdiagnose |

## Market-Objekt

| Feld | Typ | Beschreibung |
|-------|------|-------------|
| `type` | string | Markttyp-Bezeichner (z. B. `moneyline`, `point_spread`, `total_points`) |
| `selection_count` | integer | Anzahl der unterschiedlichen Auswahlmöglichkeiten innerhalb dieses Marktes |
| `book_count` | integer | Anzahl der Sportsbooks, die diesen Markt anbieten |
| `selections` | array | Liste der [Selection-Objekte](#selection-object) für diesen Markt |

## Selection-Objekt

| Feld | Typ | Beschreibung |
|-------|------|-------------|
| `name` | string | Name der Auswahl (z. B. `Boston Celtics`, `Over`, `LeBron James`) |
| `type` | string | Typ der Auswahl (z. B. `home`, `away`, `over`, `under`) |
| `line` | number | Wert der Punktespanne (Spread) oder Gesamtlinie. Wird für Märkte ohne Linien (z. B. Moneyline) weggelassen. |
| `book_count` | integer | Anzahl der Sportsbooks, die diese Auswahl anbieten |

```javascript
const eventId = 'evt_nba_bos_lal_20260208';

  `https://api.sharpapi.io/api/v1/events/${eventId}/markets`,
  { headers: { 'X-API-Key': 'YOUR_API_KEY' } }

// List markets and their selections
for (const market of data) {
  console.log(`${market.type}: ${market.book_count} books, ${market.selection_count} selections`);
  for (const sel of market.selections) {
    console.log(`  ${sel.name} (${sel.type}): ${sel.book_count} books${sel.line !== undefined ? `, line ${sel.line}` : ''}`);
  }
}
```

```python
import requests

event_id = 'evt_nba_bos_lal_20260208'
response = requests.get(
    f'https://api.sharpapi.io/api/v1/events/{event_id}/markets',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)
result = response.json()

for market in result['data']:
    print(f"{market['type']}: {market['book_count']} books, {market['selection_count']} selections")
    for sel in market['selections']:
        line_info = f", line {sel['line']}" if 'line' in sel else ''
        print(f"  {sel['name']} ({sel['type']}): {sel['book_count']} books{line_info}")
```

## Anwendungsfälle

- **Marktentdeckung** -- Ermitteln Sie, welche Markttypen verfügbar sind, bevor Sie Quoten anfordern, und reduzieren Sie so unnötige API-Aufrufe
- **Sportsbook-Abdeckung** -- Sehen Sie, welche Sportsbooks die einzelnen Märkte abdecken, um Vergleichsmöglichkeiten zu identifizieren
- **Vorbereitung für Line Shopping** -- Identifizieren Sie Märkte, die von mehreren Sportsbooks angeboten werden, um die beste Linie zu finden
- **Verfügbarkeit von Spielerwetten** -- Prüfen Sie, ob Player-Prop-Märkte verfügbar sind (oft nur bei bestimmten Sportsbooks und näher am Spielbeginn)

<Callout type="info">
Verwenden Sie diesen Endpoint, um die Marktverfügbarkeit zu prüfen, bevor Sie [Event Odds](/de/api-reference/events-odds) mit einem `market`-Filter aufrufen. Dies vermeidet die Verschwendung von API-Aufrufen für Markttypen, die für ein bestimmtes Event nicht existieren.
</Callout>
