# Sportsbooks

Rufen Sie eine Liste der verfügbaren Sportsbooks und ihres aktuellen Status ab, einschließlich der Information, welche Stufe für den Zugriff auf die Daten jedes Buchmachers erforderlich ist.

## Endpoints

```
GET /api/v1/sportsbooks
```

<Callout type="info">
Erfordert einen API-Schlüssel. Auf allen Stufen verfügbar (einschließlich Free). Nicht authentifizierte Anfragen geben `401` zurück.
</Callout>

const response = await fetch('https://api.sharpapi.io/api/v1/sportsbooks', {

## Schema des Sportsbook-Objekts

| Feld | Typ | Beschreibung |
|-------|------|-------------|
| `id` | string | Eindeutiger Bezeichner (in API-Filtern verwenden) |
| `name` | string | Interner Name (identisch mit `id`) |
| `display_name` | string | Menschenlesbarer Name |
| `has_live_odds` | boolean | Unterstützt Live-/In-Play-Quoten |
| `has_player_props` | boolean | Unterstützt Spieler-Prop-Märkte |
| `regions` | string[] | Verfügbare Regionen: `US`, `UK`, `EU`, `AU`, `GLOBAL` |
| `is_sharp` | boolean | Ob der Buchmacher als Sharp-/effizienter Markt gilt |
| `status` | string | `active` oder `inactive` |
| `event_count` | integer | Anzahl der derzeit verfügbaren Events |
| `last_update` | string | ISO-8601-Zeitstempel der letzten Datenaktualisierung |
| `requires_tier` | string \| null | Erforderliche Mindeststufe: `null` (Free), `hobby`, `pro`, `sharp` |

<Callout type="warning">
`event_count` kann 0 sein und `last_update` kann `null` sein, wenn ein Sportsbook vorübergehend nicht verfügbar ist oder Datenverzögerungen auftreten.
</Callout>

## Buchzugriff nach Stufe

Das Feld `requires_tier` gibt die Mindest-Abonnementstufe an, die benötigt wird, um über die API auf die Quotendaten eines Sportsbooks zuzugreifen.

{/* AUTO:START:tier-summary */}
| Stufe | Verfügbare Buchmacher | Enthaltene Sportsbooks |
|------|-----------------|----------------------|
| **Free** | 23 | Kalshi, BetMGM, DraftKings, Bet365 und 19 weitere |
| **Hobby** | 23 | Wie bei Free |
| **Pro** | 23 | Wie bei Hobby |
| **Sharp** | 26 | Alle verfügbaren Sportsbooks |
| **Enterprise** | 26 | Alle verfügbaren Sportsbooks |
{/* AUTO:END:tier-summary */}

<Callout type="warning">
**Pinnacle**, **Circa Sports**, **Bookmaker**, **ProphetX** und **Betfair** erfordern **Sharp-Stufe oder höher**. Pinnacle ist ein Sharp Book, dessen effiziente Linien als Referenz für +EV-Berechnungen verwendet werden. Circa Sports ist ein weiterer bekannter Sharp Book mit bewusst engen Linien und einem fokussierten Prop-Angebot. Anfragen für Sharp-Book-Quoten auf der Free-, Hobby- oder Pro-Stufe geben einen `403 tier_restricted`-Fehler zurück.
</Callout>

### `requires_tier` verstehen

| Wert | Bedeutung |
|-------|---------|
| `null` | Auf allen Stufen verfügbar, einschließlich Free |
| `"hobby"` | Erfordert Hobby-Stufe oder höher |
| `"pro"` | Erfordert Pro-Stufe oder höher |
| `"sharp"` | Erfordert Sharp- oder Enterprise-Stufe |

## Sharp- vs. Soft-Buchmacher

### Sharp Books
- **Niedrigerer Vig (Juice)** -- engere Margen bedeuten bessere Quotengenauigkeit
- **Effiziente Linien** -- werden als Referenz für +EV- und Fair-Value-Berechnungen verwendet
- **Nicht in allen Regionen verfügbar** -- Pinnacle ist in den USA nicht lizenziert
- Beispiele: Pinnacle, Circa Sports, Bookmaker

### Soft Books
- **Höherer Vig** -- breitere Margen schaffen mehr Spielraum für +EV-Möglichkeiten
- **Mehr Promotionen** -- Boni, Boosts und Gratiswetten
- **Langsamere Linienbewegung** -- können hinter Sharp-Marktbewegungen zurückbleiben
- **Wo Wert gefunden wird** -- die Lücke zwischen Soft-Book-Quoten und Sharp-Referenz erzeugt +EV
- Beispiele: DraftKings, FanDuel, BetMGM, Caesars

<Callout type="info">
Sharp Books wie Pinnacle setzen schnell effiziente Linien. Soft Books bleiben oft zurück und schaffen Zeitfenster, in denen ihre Quoten einen positiven Erwartungswert im Vergleich zur Sharp-Referenz bieten. Dies ist die Grundlage der +EV-Wettstrategie.
</Callout>

<Callout type="info">
**Erstellen Sie ein Live-Wett-Tool?** DraftKings und FanDuel haben `has_live_odds: true`, setzen jedoch häufig Märkte während Punktespielen und Schlüsselmomenten aus. Wenn Sie eine kontinuierliche Live-Abdeckung benötigen, ist Pinnacle (Sharp-Stufe) die zuverlässige Referenz. Siehe [Live vs. Pre-Match](/de/concepts/live-vs-prematch) für eine vollständige Übersicht über das Buchverhalten während des Live-Spiels.
</Callout>
