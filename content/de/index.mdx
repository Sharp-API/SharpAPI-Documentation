# SharpAPI — Novig API & Sportwettquoten

SharpAPI ist eine Echtzeit-Aggregationsplattform für Sportwettquoten, die Folgendes bietet:

- **Echtzeit-Quoten** – Quotenaktualisierungen im Sekundenbruchteil von wichtigen Sportwettenanbietern via REST oder SSE-Streaming
- **+EV-Erkennung** – Automatische Berechnung des Erwartungswerts unter Verwendung von Sharp-Book-Referenzen
- **Arbitrage-Benachrichtigungen** – Cross-Book-Arbitrage-Erkennung mit optimaler Einsatzberechnung
- **Vig-freie Quoten** – Faire Quoten ohne Buchmacher-Marge, mit Pinnacle als Sharp-Referenz
- **SSE-Streaming** – Server-Sent Events für Echtzeit-Push-Benachrichtigungen

## Basis-URL

```
https://api.sharpapi.io/api/v1
```

<Callout type="info">
Alle API-Anfragen sollten `api.sharpapi.io` als Basis-URL verwenden.
</Callout>

## Authentifizierung

Übergeben Sie Ihren API-Schlüssel mit einer der folgenden Methoden:

```bash
# Header (empfohlen)
curl -H "X-API-Key: sk_live_xxx" https://api.sharpapi.io/api/v1/odds

# Bearer-Token
curl -H "Authorization: Bearer sk_live_xxx" https://api.sharpapi.io/api/v1/odds

# Query-Parameter (für SSE)
curl "https://api.sharpapi.io/api/v1/stream?api_key=sk_live_xxx"
```

Weitere Informationen finden Sie unter [Authentifizierung](/de/authentication).
