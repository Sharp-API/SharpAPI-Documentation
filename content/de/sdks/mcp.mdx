# MCP-Server (KI-Assistenten)

<Callout type="info">
SharpAPI enthält einen integrierten MCP-Server unter `https://mcp.sharpapi.io/mcp` — verbinden Sie Claude Desktop, Claude Code oder einen beliebigen MCP-kompatiblen KI-Assistenten, um Live-Quoten und Möglichkeiten über natürliche Sprache abzufragen.
</Callout>

## Was ist MCP?

Das [Model Context Protocol](https://modelcontextprotocol.io) (MCP) ermöglicht es KI-Assistenten, externe Tools direkt aufzurufen. Anstatt Daten aus unserer API in einen Chat zu kopieren, kann Ihr KI-Assistent SharpAPI in Echtzeit abfragen:

> *„Finde die besten +EV-Möglichkeiten in der NBA gerade jetzt"*
>
> *„Vergleiche die Moneyline-Quoten für das Lakers-Spiel über alle Buchmacher hinweg"*
>
> *„Welche Arbitrage-Möglichkeiten gibt es im Fußball?"*

## Einrichtung

Fügen Sie SharpAPI zu Ihrer MCP-Client-Konfiguration hinzu:

<Tabs items={['Claude Desktop', 'Claude Code', 'Andere MCP-Clients']}>
<Tabs.Tab>

Bearbeiten Sie `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) oder `%APPDATA%\Claude\claude_desktop_config.json` (Windows):

```json
{
  "mcpServers": {
    "sharpapi": {
      "type": "streamable-http",
      "url": "https://mcp.sharpapi.io/mcp",
      "headers": {
        "X-API-Key": "sk_live_..."
      }
    }
  }
}
```

Starten Sie Claude Desktop nach dem Speichern neu.

</Tabs.Tab>
<Tabs.Tab>

Fügen Sie dies zu Ihren Claude Code MCP-Einstellungen hinzu:

```json
{
  "mcpServers": {
    "sharpapi": {
      "type": "streamable-http",
      "url": "https://mcp.sharpapi.io/mcp",
      "headers": {
        "X-API-Key": "sk_live_..."
      }
    }
  }
}
```

</Tabs.Tab>
<Tabs.Tab>

Jeder MCP-Client, der den Streamable HTTP-Transport unterstützt, kann eine Verbindung herstellen:

- **Endpoint:** `POST https://mcp.sharpapi.io/mcp`
- **Auth:** `X-API-Key`-Header, `Authorization: Bearer`-Header oder `api_key`-Query-Parameter
- **Protokoll:** JSON-RPC 2.0 über HTTP (MCP Streamable HTTP-Transport)
- **Version:** `2025-03-26`

</Tabs.Tab>
</Tabs>

## Verfügbare Tools

Der MCP-Server stellt 17 Tools bereit, die der REST-API von SharpAPI zugeordnet sind:

### Referenzdaten

| Tool | Beschreibung |
|------|-------------|
| `get_sports` | Liste aller Sportarten mit Event- und Live-Anzahl |
| `get_leagues` | Liste der Ligen, filterbar nach Sportart |
| `get_sportsbooks` | Liste aller Buchmacher mit Metadaten |
| `get_markets` | Liste aller Markttypen (Moneyline, Spread, Total, Player Props…) |

### Quoten & Events

| Tool | Beschreibung | Min. Tier |
|------|-------------|----------|
| `get_events` | Events nach Sportart, Liga oder Teamname suchen | Alle |
| `get_odds` | Aktuelle Quoten mit Filtern abrufen | Alle |
| `get_odds_delta` | Inkrementelle Synchronisierung — nur Quoten, die sich seit einem Zeitstempel geändert haben | Alle |
| `get_best_odds` | Beste Quoten über alle Buchmacher hinweg für jede Auswahl | Alle |
| `compare_odds` | Direkter Quotenvergleich für ein Event | Alle |
| `get_closing_lines` | Erfasste Closing Lines für CLV-Analyse | Pro |

### Futures

| Tool | Beschreibung | Min. Tier |
|------|-------------|----------|
| `get_futures_odds` | Saisonübergreifende Märkte (Meisterschaft, MVP, Saisonsiege) | Pro |
| `compare_futures_lines` | Buchmacherübergreifender Vergleich auf einem Futures-Markt | Pro |

### Möglichkeiten

| Tool | Beschreibung | Min. Tier |
|------|-------------|----------|
| `find_ev_opportunities` | +EV-Wetten mit Pinnacle als Referenz | Pro |
| `find_arbitrage` | Buchmacherübergreifende Arbitrage mit Einsatzberechnung | Hobby |
| `find_middles` | Middle-Möglichkeiten (beide Seiten gewinnen) | Pro |
| `find_low_hold` | Engste Linien über Buchmacher hinweg | Alle |

### Konto

| Tool | Beschreibung |
|------|-------------|
| `get_account_info` | Ihr Tier, Rate Limits und Funktionen |

## Buchmacherzugriff nach Tier

Quoten, beste Quoten und Vergleichsergebnisse werden automatisch auf die Buchmacher gefiltert, auf die Ihr Tier zugreifen kann:

| Tier | Buchmacher | Beispiele |
|------|-------|---------|
| **Free** | 2 | DraftKings, FanDuel |
| **Hobby** | 5 | + BetMGM, Caesars, Fanatics |
| **Pro** | 15 | + Pinnacle, Bet365, BetOnline, ... |
| **Sharp** | Alle 32 | Jeder Buchmacher |
| **Enterprise** | Alle 32 | + Futures, Historical, Gamestate |

## Rate Limits

Jeder MCP-Tool-Aufruf zählt als eine API-Anfrage gegen das Rate Limit Ihres Tiers. Der initiale Handshake (initialize + tools/list) kostet 2-3 Anfragen — dies geschieht einmal pro Sitzung.

| Tier | Anfragen/Min. |
|------|-------------|
| Free | 12 |
| Hobby | 120 |
| Pro | 300 |
| Sharp | 1.000 |
| Enterprise | 10.000 |

Verwenden Sie `get_account_info`, um Ihre aktuelle Nutzung und Limits zu überprüfen.

## Beispielkonversationen

Sobald die Verbindung hergestellt ist, können Sie Ihrem KI-Assistenten Fragen wie diese stellen:

- *„Welche Sportarten haben gerade jetzt Live-Events?"*
- *„Zeige mir die NBA-Quoten für die heutigen Spiele auf DraftKings"*
- *„Finde Arbitrage-Möglichkeiten im Fußball mit mindestens 1 % Gewinn"*
- *„Vergleiche die Quoten für das Chiefs-Spiel über alle Buchmacher hinweg"*
- *„Wie ist mein aktueller API-Tier und wie viele Anfragen habe ich noch übrig?"*

Die KI ruft automatisch die entsprechenden SharpAPI-Tools auf und präsentiert die Ergebnisse in einem lesbaren Format.
