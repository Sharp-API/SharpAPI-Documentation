# SharpAPI — API de Novig y cuotas de apuestas deportivas

SharpAPI es una plataforma de agregación de cuotas de apuestas deportivas en tiempo real que proporciona:

- **Cuotas en tiempo real** - Actualizaciones de cuotas en menos de un segundo desde las principales casas de apuestas mediante REST o streaming SSE
- **Detección de +EV** - Cálculo automático del valor esperado utilizando casas sharp como referencia
- **Alertas de arbitraje** - Detección de arbitraje entre casas con cálculo óptimo de apuestas
- **Cuotas sin vig** - Cuotas justas con el margen del corredor eliminado, usando Pinnacle como referencia sharp
- **Streaming SSE** - Server-Sent Events para notificaciones push en tiempo real

## URL base

```
https://api.sharpapi.io/api/v1
```

<Callout type="info">
Todas las solicitudes a la API deben usar `api.sharpapi.io` como URL base.
</Callout>

## Endpoints principales

| Endpoint | Descripción | Plan |
|----------|-------------|------|
| `GET /odds` | Snapshot de cuotas actuales | Todos |
| `GET /odds/best` | Mejores cuotas entre casas | Todos |
| `GET /events` | Eventos y calendario | Todos |
| `GET /opportunities/ev` | Oportunidades de +EV | Pro+ |
| `GET /opportunities/arbitrage` | Oportunidades de arbitraje | Hobby+ |
| `GET /opportunities/low_hold` | [Líneas de low hold](/es/api-reference/opportunities-low-hold) | Pro+ |
| `GET /stream` | Stream de cuotas en tiempo real | Complemento |

## Autenticación

Envía tu API key usando uno de estos métodos:

```bash
# Header (recomendado)
curl -H "X-API-Key: sk_live_xxx" https://api.sharpapi.io/api/v1/odds

# Bearer token
curl -H "Authorization: Bearer sk_live_xxx" https://api.sharpapi.io/api/v1/odds

# Query parameter (para SSE)
curl "https://api.sharpapi.io/api/v1/stream?api_key=sk_live_xxx"
```

Consulta [Autenticación](/es/authentication) para más información.
