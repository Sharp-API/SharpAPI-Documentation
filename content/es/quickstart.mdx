# Inicio rápido

Empieza a usar SharpAPI en 2 minutos.

## URL base

```
https://api.sharpapi.io/api/v1
```

<Callout type="warning">
Utiliza siempre `api.sharpapi.io` —no `sharpapi.io`— para las solicitudes a la API.
</Callout>

<Steps>

### Obtén tu clave de API

1. **Regístrate** - Crea una cuenta gratuita en [sharpapi.io/sign-up](https://sharpapi.io/sign-up)
2. **Obtén tu clave** - Copia tu clave de API desde el panel de control. Comienza con `sk_live_`
3. **Realiza tu primera llamada** - Utiliza los ejemplos siguientes para obtener cuotas en directo

### Tu primera llamada a la API

<Tabs items={['cURL', 'JavaScript', 'Python', 'TypeScript']}>
  <Tabs.Tab>
```bash
curl -X GET "https://api.sharpapi.io/api/v1/odds" \
  -H "X-API-Key: YOUR_API_KEY"
```
  </Tabs.Tab>
  <Tabs.Tab>
```javascript
const response = await fetch('https://api.sharpapi.io/api/v1/odds', {
  headers: {
    'X-API-Key': 'YOUR_API_KEY'
  }
});

const { data, meta } = await response.json();
console.log(`Found ${meta.count} odds`);
```
  </Tabs.Tab>
  <Tabs.Tab>
```python
import requests

response = requests.get(
    'https://api.sharpapi.io/api/v1/odds',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)

data = response.json()
print(f"Found {data['meta']['count']} odds")
```
  </Tabs.Tab>
  <Tabs.Tab>
```typescript
import { SharpApiClient } from '@sharpapi/client';

const client = new SharpApiClient('YOUR_API_KEY');
const { data, meta } = await client.getOdds();

console.log(`Found ${meta.count} odds`);
```
  </Tabs.Tab>
</Tabs>

## Filtra tus resultados

Añade parámetros de consulta para acotar los resultados:

```bash
# Obtener cuotas de la NBA solo de DraftKings
curl "https://api.sharpapi.io/api/v1/odds?league=nba&sportsbook=draftkings" \
  -H "X-API-Key: YOUR_API_KEY"

# Obtener solo eventos en directo
curl "https://api.sharpapi.io/api/v1/odds?live=true" \
  -H "X-API-Key: YOUR_API_KEY"

# Obtener solo los mercados principales (moneyline, spread, total)
curl "https://api.sharpapi.io/api/v1/odds?market=moneyline,spread,total" \
  -H "X-API-Key: YOUR_API_KEY"
```

## Streaming en tiempo real

Para obtener las actualizaciones más rápidas, utiliza el streaming SSE (requiere el complemento WebSocket):

```javascript
const eventSource = new EventSource(
  'https://api.sharpapi.io/api/v1/stream?channel=all&api_key=YOUR_KEY'
);

eventSource.addEventListener('connected', (e) => {
  const { stream_id } = JSON.parse(e.data);
  console.log('Stream connected:', stream_id);
});

eventSource.addEventListener('snapshot', (e) => {
  const { odds } = JSON.parse(e.data);
  console.log('Initial snapshot:', odds.length, 'odds');
});

eventSource.addEventListener('odds:update', (e) => {
  const update = JSON.parse(e.data);
  console.log('Update from:', update.sportsbook);
});

eventSource.addEventListener('ev:detected', (e) => {
  const opp = JSON.parse(e.data);
  console.log(`+EV found: ${opp.selection} at ${opp.ev_percentage}% EV`);
});
```

## Problemas comunes

<Callout type="error" emoji="!">
**Error: 401 Unauthorized**

Asegúrate de utilizar la clave de API correcta y de que esté correctamente formateada en la cabecera.
</Callout>

<Callout type="error" emoji="!">
**Error: 429 Rate Limited**

Has superado tu límite de tasa. Espera al tiempo de restablecimiento indicado en `X-RateLimit-Reset` o actualiza tu plan.
</Callout>

<Callout type="error" emoji="!">
**Error: 403 Tier Restricted**

Este endpoint requiere un nivel superior. Consulta la página de [Precios](/es/pricing) para ver las funcionalidades de cada nivel.
</Callout>
