# Esportes

Obtenha uma lista de todos os esportes disponíveis com suas ligas e contagens de eventos.

## Endpoints

```
GET /api/v1/sports
```

<Callout type="info">
Requer uma API key. Disponível em todos os planos (Free incluído). Requisições não autenticadas retornam `401`.
</Callout>

## Schema do Objeto Sport

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | string | Identificador do esporte (minúsculas, usado em filtros da API) |
| `name` | string | Nome do esporte legível para humanos |
| `leagues` | string[] | IDs das ligas dentro deste esporte |
| `event_count` | integer | Total de eventos atualmente disponíveis com odds |
| `live_count` | integer | Eventos atualmente ao vivo/em andamento |

## Usando Esportes em Filtros

Use o `id` do esporte como parâmetro de filtro em outros endpoints:

```bash
# Get odds for basketball only
curl "https://api.sharpapi.io/api/v1/odds?sport=basketball" \
  -H "X-API-Key: YOUR_API_KEY"

# Get events for multiple sports
curl "https://api.sharpapi.io/api/v1/events?sport=basketball,football" \
  -H "X-API-Key: YOUR_API_KEY"
```
