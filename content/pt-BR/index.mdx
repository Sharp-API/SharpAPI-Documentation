# SharpAPI — Novig API e Odds de Apostas Esportivas

A SharpAPI é uma plataforma de agregação de odds de apostas esportivas em tempo real que oferece:

- **Odds em tempo real** - Atualizações de odds em frações de segundo das principais casas de apostas via REST ou streaming SSE
- **Detecção de +EV** - Cálculo automático de valor esperado usando referências de sharp books
- **Alertas de arbitragem** - Detecção de arbitragem entre casas com cálculos de stake ideais
- **Odds sem vig** - Odds justas com a margem da casa removida, usando a Pinnacle como referência sharp
- **Streaming SSE** - Server-Sent Events para notificações push em tempo real

## URL Base

```text
https://api.sharpapi.io/api/v1
```

<Callout type="info">
Todas as requisições à API devem usar `api.sharpapi.io` como URL base.
</Callout>

## Endpoints Principais

| Endpoint | Descrição | Plano |
|----------|-------------|------|
| `GET /odds` | Snapshot atual de odds | Todos |
| `GET /odds/best` | Melhores odds entre casas | Todos |
| `GET /events` | Eventos e programação | Todos |
| `GET /opportunities/ev` | Oportunidades de +EV | Pro+ |
| `GET /opportunities/arbitrage` | Oportunidades de arbitragem | Hobby+ |
| `GET /opportunities/low_hold` | [Linhas de low hold](/pt-BR/api-reference/opportunities-low-hold) | Pro+ |
| `GET /stream` | Stream de odds em tempo real | Complemento |

## Autenticação

Passe sua API key usando um destes métodos:

```bash
# Header (recomendado)
curl -H "X-API-Key: sk_live_xxx" https://api.sharpapi.io/api/v1/odds

# Bearer token
curl -H "Authorization: Bearer sk_live_xxx" https://api.sharpapi.io/api/v1/odds

# Parâmetro de query (para SSE)
curl "https://api.sharpapi.io/api/v1/stream?api_key=sk_live_xxx"
```

Veja [Autenticação](/pt-BR/authentication) para mais detalhes.
