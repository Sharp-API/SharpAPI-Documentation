## Rate Limiting

Todas as requisições à API são limitadas por taxa por API key. Os cabeçalhos de rate limit são incluídos em cada resposta:

```http
X-RateLimit-Limit: 300
X-RateLimit-Remaining: 299
X-RateLimit-Reset: 1703123460
```

### Resposta de Rate Limit

Quando você exceder seu rate limit, receberá uma resposta `429`:

```json
{
  "error": {
    "code": "rate_limited",
    "message": "Rate limit exceeded",
    "retry_after": 45
  }
}
```

<Callout type="info">
Use o valor `retry_after` (em segundos) para implementar backoff exponencial.
</Callout>
