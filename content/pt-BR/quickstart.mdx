# Início Rápido

Comece a usar a SharpAPI em 2 minutos.

## URL Base

```text
https://api.sharpapi.io/api/v1
```

<Callout type="warning">
Sempre use `api.sharpapi.io` - não `sharpapi.io` - para requisições à API.
</Callout>

<Steps>

### Obtenha sua API Key

1. **Cadastre-se** - Crie uma conta gratuita em [sharpapi.io/sign-up](https://sharpapi.io/sign-up)
2. **Obtenha sua chave** - Copie sua API key do dashboard. Ela começa com `sk_live_`
3. **Faça sua primeira chamada** - Use os exemplos abaixo para buscar odds ao vivo

### Sua Primeira Chamada de API

<Tabs items={['cURL', 'JavaScript', 'Python', 'TypeScript']}>
  <Tabs.Tab>
```bash
curl -X GET "https://api.sharpapi.io/api/v1/odds" \
  -H "X-API-Key: YOUR_API_KEY"
```
  </Tabs.Tab>
  <Tabs.Tab>
```javascript
const response = await fetch('https://api.sharpapi.io/api/v1/odds', {
  headers: {
    'X-API-Key': 'YOUR_API_KEY'
  }
});

const { data, meta } = await response.json();
console.log(`Found ${meta.count} odds`);
```
  </Tabs.Tab>
  <Tabs.Tab>
```python
import requests

response = requests.get(
    'https://api.sharpapi.io/api/v1/odds',
    headers={'X-API-Key': 'YOUR_API_KEY'}
)

data = response.json()
print(f"Found {data['meta']['count']} odds")
```
  </Tabs.Tab>
  <Tabs.Tab>
```typescript
import { SharpApiClient } from '@sharpapi/client';

const client = new SharpApiClient('YOUR_API_KEY');
const { data, meta } = await client.getOdds();

console.log(`Found ${meta.count} odds`);
```
  </Tabs.Tab>
</Tabs>

## Filtre seus Resultados

Adicione parâmetros de query para refinar os resultados:

```bash
# Obtenha odds da NBA apenas da DraftKings
curl "https://api.sharpapi.io/api/v1/odds?league=nba&sportsbook=draftkings" \
  -H "X-API-Key: YOUR_API_KEY"

# Obtenha apenas eventos ao vivo
curl "https://api.sharpapi.io/api/v1/odds?live=true" \
  -H "X-API-Key: YOUR_API_KEY"

# Obtenha apenas mercados principais (moneyline, spread, total)
curl "https://api.sharpapi.io/api/v1/odds?market=moneyline,spread,total" \
  -H "X-API-Key: YOUR_API_KEY"
```

## Streaming em Tempo Real

Para as atualizações mais rápidas, use streaming SSE (requer o add-on de WebSocket):

```javascript
const eventSource = new EventSource(
  'https://api.sharpapi.io/api/v1/stream?channel=all&api_key=YOUR_KEY'
);

eventSource.addEventListener('connected', (e) => {
  const { stream_id } = JSON.parse(e.data);
  console.log('Stream connected:', stream_id);
});

eventSource.addEventListener('snapshot', (e) => {
  const { odds } = JSON.parse(e.data);
  console.log('Initial snapshot:', odds.length, 'odds');
});

eventSource.addEventListener('odds:update', (e) => {
  const update = JSON.parse(e.data);
  console.log('Update from:', update.sportsbook);
});

eventSource.addEventListener('ev:detected', (e) => {
  const opp = JSON.parse(e.data);
  console.log(`+EV found: ${opp.selection} at ${opp.ev_percentage}% EV`);
});
```

## Problemas Comuns

<Callout type="error" emoji="!">
**Erro: 401 Unauthorized**

Certifique-se de estar usando a API key correta e que ela está formatada adequadamente no header.
</Callout>

<Callout type="error" emoji="!">
**Erro: 429 Rate Limited**

Você excedeu seu rate limit. Aguarde o tempo de reset indicado em `X-RateLimit-Reset` ou faça upgrade do seu plano.
</Callout>

<Callout type="error" emoji="!">
**Erro: 403 Tier Restricted**

Este endpoint requer um tier superior. Confira a página de [Preços](/pt-BR/pricing) para ver os recursos de cada tier.
</Callout>
