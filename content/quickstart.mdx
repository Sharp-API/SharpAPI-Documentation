# Quick Start

Get started with SharpAPI in 2 minutes.

## Base URL

```
https://api.sharpapi.io/api/v1
```

<Callout type="warning">
Always use `api.sharpapi.io` - not `sharpapi.io` - for API requests.
</Callout>

<Steps>

### Get Your API Key

1. **Sign Up** - Create a free account at [sharpapi.io/signup](https://sharpapi.io/signup)
2. **Get Your Key** - Copy your API key from the dashboard. It starts with `sk_live_` or `sk_test_`
3. **Make Your First Call** - Use the examples below to fetch live odds

## Filter Your Results

Add query parameters to narrow down results:

```bash
# Get NBA odds from DraftKings only
curl "https://api.sharpapi.io/api/v1/odds?league=nba&sportsbook=draftkings" \
  -H "X-API-Key: YOUR_API_KEY"

# Get only live events
curl "https://api.sharpapi.io/api/v1/odds?live=true" \
  -H "X-API-Key: YOUR_API_KEY"

# Get main markets only (moneyline, spread, total)
curl "https://api.sharpapi.io/api/v1/odds?market=moneyline,spread,total" \
  -H "X-API-Key: YOUR_API_KEY"
```

## Real-time Streaming

For the fastest updates, use SSE streaming (requires WebSocket add-on):

```javascript
const eventSource = new EventSource(
  'https://api.sharpapi.io/api/v1/stream?channel=all&api_key=YOUR_KEY'
);

eventSource.addEventListener('connected', (e) => {
  const { stream_id } = JSON.parse(e.data);
  console.log('Stream connected:', stream_id);
});

eventSource.addEventListener('snapshot', (e) => {
  const { odds } = JSON.parse(e.data);
  console.log('Initial snapshot:', odds.length, 'odds');
});

eventSource.addEventListener('odds:update', (e) => {
  const update = JSON.parse(e.data);
  console.log('Update from:', update.sportsbook);
});

eventSource.addEventListener('ev:detected', (e) => {
  const opp = JSON.parse(e.data);
  console.log(`+EV found: ${opp.selection} at ${opp.ev_percent}% EV`);
});
```

## Common Issues

<Callout type="error" emoji="!">
**Error: 401 Unauthorized**

Make sure you're using the correct API key and it's properly formatted in the header.
</Callout>

<Callout type="error" emoji="!">
**Error: 429 Rate Limited**

You've exceeded your rate limit. Wait for the reset time indicated in `X-RateLimit-Reset` or upgrade your plan.
</Callout>

<Callout type="error" emoji="!">
**Error: 403 Tier Restricted**

This endpoint requires a higher tier. Check the [Pricing](/pricing) page for tier features.
</Callout>
